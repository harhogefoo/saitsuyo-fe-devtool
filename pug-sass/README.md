# さいつよのFE開発
- pug ... HTMLテンプレートエンジン
- sass ... css
- autoprefixer ... ブラウザごとに異なるcssを自動付与
- plumber ... エラーが発生してもwatchを止めないようにする
- notify ... デスクトップ通知
- browser-sync ... ブラウザをリロードする
- gulp ... タスクランナー

## 環境構築
```
$ yarn
```

## 実行方法
```
$ gulp
```
