# Pug + sass + gulp
## 環境構築
```
$ yarn global add pug
$ yarn global add gulp
$ yarn add gulp-pug
$ yarn add gulp-sass
$ pug --version
```

## 実行方法
```
$ gulp
$ gulp watch
```
